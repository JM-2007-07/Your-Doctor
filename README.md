# Your-Doctor

Современный лендинг с минималистичным дизайном и аккуратным интерфейсом.  
Проект выполнен по макету из Figma и подходит для портфолио или практики веб-разработки.

В дизайне используются большие заголовки, чистая типографика, современные карточки и адаптивный layout.  
Сайт хорошо смотрится как на компьютерах, так и на мобильных устройствах.
